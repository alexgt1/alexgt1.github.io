Time for a new project

At this point in time I'd finished working on the THETA website and was close to finishing off the Polytech BIT site as well.

I got an email from Adon wondering if I would be interested in putting together a website for the Home of St Barnabas trust as there were still a few weeks remaining in the term. I thought it sounded like a good project as they also had content and images to provide so I would be able to get it up and running quickly.

Design Inspiration

Spent todays project class searching for similar sites to get an idea of how the website should be laid out. One thing that surprised me was the lack of up to date Retirement Home websites.

Page Hierachy and Design Components

Having now finished the home page I spent todays class working on the structure of the sub pages.

I prioritised simplicity and decided to go for a two column layout. The main content of the page was to be positioned in the left hand column which is wider than the right and takes up the majority of the page. The right hand column would be used for important information such as contact detals, pricing and business hours. I thought it would be a good idea to use bootstraps card component for the various segments in this column to distinguish from the main content and also catch the users attention. As well as this I added a breadcrumb navigation to the top of the main body so that the user knew what page they were on and how they got there.

Implementing Google map

Owl Carousel Javascript Plugin

I decided a slider/carousel would be a good feature for the homepage. Sliders are a good way of displaying a large amount of information without taking up too much space. This slider will be used to showcase some of the images that Lynette sent through. As a lot of these images are low quality it makes sense to use a component such as this as it will help to hide the pixelation.

From browsing varius sites and viewing source code Owl Carousel seemed like a good choice for a slider. As well as being popular it is fully responsive and offers touch and drag support.

Besides getting some skewed images initially, the implimentation was fairly straight forward. All I had to do was update the number of images I'd like to be displayed for each of the dimensions.

Image Collections with Jekyll

Due to the number of images I'd accumulated I decided against inserting them all using the 'img' tag and instead took a more modular approach.

Having built the THETA website using Jekyll I thought I'd refer back to what I'd learnt and use collections to organise the slider images. Collections are a great way of posting groups of content that share similarities, this includes articles or specific information you'd like to include in a dropdown menu.

More Content

Adon forwarded me an email he recieved from Lynette with some additional images for the site so I spent a bit of time adding them in. At this point the site is more a less finished so I'll spend the next few classes refining the details.

Combating poor image quality with RGBA and Overlays

Ran into some trouble with the hero images for the site. The images provided worked well in the carousel but when displayed at larger dimensions would lose their quality. As a result I decided to add a blue overlay to the images to hide some of the pixelation. Overall they turned out rather well as I was able to incoroprate the blue used in the logo with the image overlay.
